# Enterprise Network Design using Cisco Packet Tracer

## Project Overview
This project simulates an enterprise network implemented using Cisco Packet Tracer
as part of a Computer Networks university course.

## Implemented Features
- Subnetting and IP address assignment
- Connectivity testing using ping
- Device security (passwords & Telnet)
- DHCP for Secretary network
- Dynamic routing using OSPF
- HTTP Server with DNS
- FTP Server

## Tools Used
- Cisco Packet Tracer
